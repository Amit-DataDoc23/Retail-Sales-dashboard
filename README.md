# 📊 Retail Sales Dashboard (Power BI)

## Overview
The **Retail Sales Dashboard** is an interactive business intelligence solution built on **Power BI** to analyze and visualize retail performance across multiple dimensions. It provides actionable insights into sales, profitability, product performance, regional distribution, and customer segments.

## Features
- **Key Performance Indicators (2019–2024):**
  - Total Sales: 4.72bn
  - Total Quantity: 316K
  - Profit: 874M
  - Profit Margin: 18.49%
  - Customer Segment filter for dynamic analysis

- **Top Products Analysis:**
  Horizontal bar chart showcasing the top 10 products by sales (Desk, Washing Machine, Smartphone, Tablet, etc.).

- **Store Type & Region Distribution:**
  Donut charts highlighting sales contribution by store type (Online, Franchise, Offline) and region (North, South, East, West).

- **Geographic Insights:**
  Interactive map of India displaying total sales by state (Maharashtra, Karnataka, Tamil Nadu, Delhi, etc.).

- **Sales Trend Comparison:**
  Monthly bar chart comparing current year sales vs last year’s sales.

- **Brand Performance:**
  Tabular view of top 5 brands with quantity, sales, and profit metrics (SpinWash, StackWell, WorkWood, ComfortSeat, BeatSound).

## Tech Stack
- Power BI for dashboard creation and interactivity
- Data Modeling for relationships and measures
- DAX for calculated KPIs and metrics
